# Belanja Sakan Management System
Ini adalah Sistem Pengurusan untuk belanja harian.
Ia terdiri dari 3 bahagian utama
* user
* admin1
* admin2

### paparan utama
___
* "index" **Home**
* "login" **Login**
* "daftar" **Register**
* "lupapass" **Lupa Password**

* Belian

  * "baru" **Tambah Belian**
